Don't know how to connect to HTB VPN & SSH? Click here.

Download the VPN configuration file from HTB Academy and run the following command in the same directory:

sudo openvpn academy-regular.ovpn # Make sure OpenVPN is installed.

Command Breakdown

  • sudo → Runs the command with administrator (root) privileges.
  • openvpn → Starts the OpenVPN client.
  • academy-regular.ovpn → The VPN configuration file that contains the settings required to connect to the HTB Academy VPN.

Once the VPN connection is established, connect to the spawned target machine using SSH:

ssh htb-student@<ip-address> # Use password: HTB_@cademy_stdnt!

Command Breakdown

  • ssh → Starts a Secure Shell (SSH) session to connect to a remote system.
  • htb-student → The username used to log in.
  • @ → Separates the username from the target host.
  • <ip-address> → The IP address of the target machine.

After successfully authenticating, you will be presented with the terminal of the target machine.

Find out the machine hardware name and submit it as the answer.

The uname command displays information about the operating system and kernel. Using the -m option displays the machine's hardware architecture.

uname -m

Command Breakdown

  • uname → Displays system information.
  • -m → Displays the machine hardware architecture.

Submit the output as the answer.

What is the path to htb-student's home directory?

Environment variables store information about the current user and system. The HOME environment variable contains the absolute path to the current user's home directory.

echo $HOME

Command Breakdown

  • echo → Displays the value passed to it.
  • $HOME → Stores the path to the current user's home directory.

Note: If you are currently inside your home directory, you can also use the pwd command to display its path.

What is the name of the network interface that MTU is set to 1500?

The ifconfig command displays information about the network interfaces configured on the system, including their IP addresses, status, and MTU (Maximum Transmission Unit).

ifconfig

Command Breakdown

  • ifconfig → Displays information about the system's network interfaces.

Manually look for the interface whose configuration contains mtu 1500.

Alternatively, use the following command to filter the output:

ifconfig | grep "mtu 1500"

Command Breakdown

  • ifconfig → Displays information about the system's network interfaces.
  • | → Sends the output of the first command to the next command.
  • grep → Searches for lines containing the specified text.
  • "mtu 1500" → Searches for lines that contain the text mtu 1500.

The name of the network interface appears at the beginning of the matching line.

Note: ifconfig is considered a legacy networking tool on many modern Linux distributions. The ip command is the recommended replacement, although ifconfig is still commonly encountered in labs, CTFs, and older systems.
