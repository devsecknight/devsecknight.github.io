Don't know how to connect to HTB VPN & RDP? Click here.

Download the VPN configuration file from HTB Academy and run the following command in the same directory:

sudo openvpn academy-regular.ovpn # Make sure OpenVPN is installed.

Command Breakdown

  • sudo → Runs the command with administrator (root) privileges.
  • openvpn → Starts the OpenVPN client.
  • academy-regular.ovpn → The VPN configuration file that contains the settings required to connect to the HTB Academy VPN.

Once the VPN connection is established, connect to the spawned target machine using Remote Desktop Protocol (RDP):

xfreerdp /v:<ip-address> /u:htb-student /p:'Academy_WinFun!' # Install xfreerdp if needed.

Command Breakdown

  • xfreerdp → A free and open-source RDP client for Linux.
  • /v:<ip-address> → Specifies the IP address of the target Windows machine.
  • /u:htb-student → The username used to log in.
  • /p:'HTB_@cademy_stdnt!' → The password for the user account.

After successfully authenticating, you will be presented with the desktop of the target Windows machine.

Find the Non-Standard Directory in the C Drive. Submit the contents of the flag file saved in this directory.

The objective is to identify the non-standard directory on the C: drive and retrieve the contents of the flag.txt file stored within it.

Standard directories on the C: drive typically include:

  • PerfLogs
  • Program Files
  • Program Files (x86)
  • ProgramData
  • Users
  • Windows

Any additional directories are considered non-standard and should be investigated.

To list the contents of the C: drive, run:

dir C:\

Command Breakdown

  • dir → A built-in Windows command used to display the contents of a directory.
  • C:\ → Specifies the root of the C: drive, where the directory listing will be performed.

Review the output and identify any directories that are not part of the standard Windows installation. Navigate through these directories until you locate the one containing the flag.txt file.

Once you have identified the correct directory, display the contents of the flag using:

cat C:\Academy\flag.txt

Command Breakdown

  • cat → A PowerShell alias for the Get-Content cmdlet. Reads and displays the contents of a file.
  • C:\Academy\flag.txt → Specifies the path to the file whose contents will be displayed.

The command prints the contents of the flag file to the terminal. Submit the displayed value as your answer.

Note: This task can also be completed using the Windows graphical user interface by browsing the C: drive in File Explorer.
