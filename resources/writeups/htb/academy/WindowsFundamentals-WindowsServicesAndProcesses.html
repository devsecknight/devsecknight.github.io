Don't know how to connect to HTB VPN & RDP? Click here.

Download the VPN configuration file from HTB Academy and run the following command in the same directory:

sudo openvpn academy-regular.ovpn # Make sure OpenVPN is installed.

Command Breakdown

  • sudo → Runs the command with administrator (root) privileges.
  • openvpn → Starts the OpenVPN client.
  • academy-regular.ovpn → The VPN configuration file that contains the settings required to connect to the HTB Academy VPN.

Once the VPN connection is established, connect to the spawned target machine using Remote Desktop Protocol (RDP):

xfreerdp /v:<ip-address> /u:htb-student /p:'Academy_WinFun!' # Install xfreerdp if needed.

Command Breakdown

  • xfreerdp → A free and open-source RDP client for Linux.
  • /v:<ip-address> → Specifies the IP address of the target Windows machine.
  • /u:htb-student → The username used to log in.
  • /p:'HTB_@cademy_stdnt!' → The password for the user account.

After successfully authenticating, you will be presented with the desktop of the target Windows machine.

Identify one of the non-standard update services running on the host. Submit the full name of the service executable (not the DisplayName) as your answer.

To view all running services, run:

Get-Service | Where-Object {$_.Status -eq "Running"}

Command Breakdown

  • Get-Service → Retrieves all services installed on the system.
  • | → Passes the service objects to the next command.
  • Where-Object → Filters objects based on a specified condition.
  • $_.Status → Refers to the Status property of each service.
  • -eq → Comparison operator that checks for an exact match.
  • "Running" → Filters the results to display only services that are currently running.

Review the output and identify a service that is not part of a standard Windows installation.

If the challenge requires the service executable name rather than the DisplayName, inspect the service configuration using:

sc.exe qc FoxitReaderUpdateService

Command Breakdown

  • sc.exe → The Service Control utility used to manage and query Windows services.
  • qc → Stands for Query Configuration. Displays the configuration information for the specified service.
  • FoxitReaderUpdateService → Specifies the service whose configuration will be displayed.

Locate the BINARY_PATH_NAME field in the output. The executable name at the end of this path is the value to submit as your answer.
