Don't know how to connect to HTB VPN & RDP? Click here.

Download the VPN configuration file from HTB Academy and run the following command in the same directory:

sudo openvpn academy-regular.ovpn # Make sure OpenVPN is installed.

Command Breakdown

  • sudo → Runs the command with administrator (root) privileges.
  • openvpn → Starts the OpenVPN client.
  • academy-regular.ovpn → The VPN configuration file that contains the settings required to connect to the HTB Academy VPN.

Once the VPN connection is established, connect to the spawned target machine using Remote Desktop Protocol (RDP):

xfreerdp /v:<ip-address> /u:htb-student /p:'Academy_WinFun!' # Install xfreerdp if needed.

Command Breakdown

  • xfreerdp → A free and open-source RDP client for Linux.
  • /v:<ip-address> → Specifies the IP address of the target Windows machine.
  • /u:htb-student → The username used to log in.
  • /p:'HTB_@cademy_stdnt!' → The password for the user account.

After successfully authenticating, you will be presented with the desktop of the target Windows machine.

What is the alias set for the ipconfig.exe command?

To find the alias associated with ipconfig.exe, run:

Get-Alias | findstr ipconfig

Command Breakdown

  • Get-Alias → A PowerShell cmdlet that lists all aliases available in the current PowerShell session. An alias is an alternate name assigned to a command or executable, allowing it to be invoked using a shorter or more familiar name.
  • | → The pipeline operator. Passes the output of Get-Alias as input to the next command.
  • findstr → A Windows command-line utility used to search for a specific string within text.
  • ipconfig → The search string used by findstr. Filters the output to display only entries related to ipconfig.exe.

The command displays the alias mapped to ipconfig.exe. Submit the alias as your answer.

Note: PowerShell aliases provide alternative names for commands. They do not create new commands or change their functionality.

Find the Execution Policy set for the LocalMachine scope.

To view the execution policies configured for each scope, run:

Get-ExecutionPolicy -List | findstr LocalMachine

Command Breakdown

  • Get-ExecutionPolicy → A PowerShell cmdlet that displays the execution policy configured on the system. An Execution Policy is a PowerShell security feature that determines the conditions under which scripts can be executed.
  • -List → Displays the execution policy configured for every available scope.
  • | → Passes the output of Get-ExecutionPolicy to the next command.
  • findstr → Searches the command output for a specific string.
  • LocalMachine → Filters the output to display only the execution policy configured for the LocalMachine scope.

The LocalMachine scope applies to all users on the system and stores its configuration in the local computer settings.

Submit the execution policy configured for the LocalMachine scope as your answer.
