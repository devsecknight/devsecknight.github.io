Don't know how to connect to HTB VPN & RDP? Click here.

Download the VPN configuration file from HTB Academy and run the following command in the same directory:

sudo openvpn academy-regular.ovpn # Make sure OpenVPN is installed.

Command Breakdown

  • sudo → Runs the command with administrator (root) privileges.
  • openvpn → Starts the OpenVPN client.
  • academy-regular.ovpn → The VPN configuration file that contains the settings required to connect to the HTB Academy VPN.

Once the VPN connection is established, connect to the spawned target machine using Remote Desktop Protocol (RDP):

xfreerdp /v:<ip-address> /u:htb-student /p:'Academy_WinFun!' # Install xfreerdp if needed.

Command Breakdown

  • xfreerdp → A free and open-source RDP client for Linux.
  • /v:<ip-address> → Specifies the IP address of the target Windows machine.
  • /u:htb-student → The username used to log in.
  • /p:'HTB_@cademy_stdnt!' → The password for the user account.

After successfully authenticating, you will be presented with the desktop of the target Windows machine.

What protocol discussed in this section is used to share resources on the network using Windows? (Format: case sensitive)

The Server Message Block (SMB) protocol is the standard Windows network protocol used to share files, folders, printers, and other resources between systems on a network.

What is the name of the utility that can be used to view logs made by a Windows system? (Format: 2 words, 1 space, not case sensitive)

Event Viewer is the built-in Windows utility used to view and analyze system, application, and security logs generated by the operating system and installed applications.

What is the full directory path to the Company Data share we created?

Before solving this question, create a folder and configure it as a shared folder by navigating to:

Right-click Folder → Properties → Sharing → Advanced Sharing → Check "Share this folder" → Apply

To view all active network shares on the local machine, run:

net share

Command Breakdown

  • net → A built-in Windows command-line utility used to manage and display network-related resources and services.
  • share → Displays all shared resources configured on the local machine. It can also be used to create, modify, or remove shared folders.

The command lists every shared resource along with its corresponding local directory path. Locate the Company Data share in the output and submit its full directory path as your answer.
