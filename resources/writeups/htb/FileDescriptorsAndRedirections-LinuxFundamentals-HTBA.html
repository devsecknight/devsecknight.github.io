How many files exist on the system that have the ".log" file extension?

Use the command:

find / -type f -name "*.log" 2>/dev/null | wc -l
  • find /: Searches starting from the root directory.
  • 2>/dev/null: Hides "Permission denied" errors so they don't break the counter.
  • wc -l: Counts the total number of lines outputted (one line per file).

Submit the number as the answer.

How many total packages are installed on the target system?

Use the command:

apt list --installed 2>/dev/null | grep -v "^Listing..." | wc -l
  • grep -v "^Listing...": Excludes the informational header line that apt prints at the very beginning. (Without excluding it, wc -l counts that line as a package, giving an answer off by one.)
  • wc -l: Counts the total lines.
